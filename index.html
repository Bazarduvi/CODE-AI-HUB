📖 Manual de Uso

🚀 Inicio rápido

1. Ve a ⚙️ Config y añade tu Groq API Key (gratuita en console.groq.com).
2. Vuelve a ⚡ Generador, escribe tu prompt y pulsa 🤖 Generar.
3. Copia o guarda el resultado como snippet.
4. Usa 🧪 Sandbox para probar el código en tiempo real.

🤖 Modelos disponibles

Groq (gratuito): Llama3 8B, Llama3 70B, Mixtral 8x7B — velocidad extrema.
Gemini (gratuito): Gemini 2.0 Flash — multimodal, contexto largo.
Selecciona el modelo en el desplegable del header.

📲 Instalar como PWA

Pulsa el botón 📲 Instalar del header cuando aparezca.
En iOS: Safari → Compartir → Añadir a pantalla de inicio.
En Android: Chrome → Menú → Instalar app.

📦 Empaquetar APK con PWABuilder

1. Sube tu app a un servidor HTTPS (GitHub Pages, Netlify, Vercel).
2. Ve a pwabuilder.com.
3. Introduce la URL de tu app y genera el APK.
4. Descarga el .apk y distribúyelo o súbelo a Play Store.

☁️ Google Drive

Necesitas un Client ID de Google Cloud Console y un Access Token OAuth2.
El token caduca cada hora. Renuévalo en OAuth Playground.
Scope necesario: https://www.googleapis.com/auth/drive.file

🧠 Entrenamiento IA

Compatible con Ollama, LM Studio, OpenLLM y cualquier API REST.
Los datos de entrenamiento se guardan en localStorage y se pueden sincronizar con Drive.
